# calc-service

## to run in docker
`docker-compose up`

## to run on local file system

### install python 3.7 virtual env
`virtualenv -p python3.7 dockflaskvenv`
`source dockflaskvenv/bin/activate`
`cd backend`
`FLASK_APP=calcservice.py python -m flask run`

## View

go to http://127.0.0.1:5000/
